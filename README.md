# Unit_Testing
Модульное тестирование (юнит-тестирование, Unit Testing) — это метод тестирования программного обеспечения, направленный на проверку отдельных модулей или компонентов программы. Модуль — небольшая, изолированная часть программы (функция, метод, класс или объект), выполняющая определённую задачу.
